# Projeto_esp32






1)
Para realizar a execução desse código será necessário que pegue os códigos dentro da pasta esp e salva-los em Micropython_advice no thonny para o funcionamento da esp32 e para o funcionamento do display.






2)
Após adicionar os códigos da eps32 em advices, será necessário executar o xampp executando o Mysql e o apache.






3)
Depois baixar o banco de dados forecido na pasta inicial (Projeto_esp32) e importar para o phpadmin do xampp.






4)
Após exportar o banco abra a pasta app no vscode e execute o código app.py e o código scanner_bridge_global.py.(O código scanner_bridge_global.py serve para a leitura do leitor de código de barras).
